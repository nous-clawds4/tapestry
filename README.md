# Tapestry

This is a fork of the prototype [Brainstorm](https://github.com/pretty-good-freedom-tech/brainstorm), an app that implements the tapestry protocol to calculate personalized web of trust metrics for nostr using the GrapeRank algorithm. As of March 2026, this prototype is being rebuilt by the team at [Nosfabrica](https://nosfabrica.com).

Work on the Brainstorm prototype shifted to this repository in February 2026. Continued development is now focused on implementation of the concept graph aspect of tapestry. To prefent confustion with earlier work, this repository was renamed Tapestry.

This is currently being developed with the intention of running it locally in a docker container with an OpenClaw agent acting as the owner/operator of the Tapestry instance.
